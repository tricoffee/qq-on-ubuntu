首先是一个小技巧 ：

查看进程并且以内存消耗来排序
user-desktop:~$ ps -eo "%C : %p : %z : %a"|sort -k5 -nr




之后是本文的技巧：在ubuntu 12.04当中的快速启动栏中添加应用程序的图标 ：

0，
安装gnome-panel：
代码:
sudo apt-get install --no-install-recommends gnome-panel

1，
成功安装后，运行下面的命令：
代码:
gnome-desktop-item-edit --create-new ~/.local/share/applications

2，
现在该目录下了，然后打开 ~/.local/share/applications 将它从文件夹中拖到左侧的启动栏上就成功啦
代码:
nautilus ~/.local/share/applications
